# Toaster Bot

Это Telegram-бот на Python, который симулирует «живого» тостера с характером, тревожностью и чувством юмора.

## Возможности:
- Отвечает на команды
- Сохраняет логи общения
- Меняет настроение
- Обладает зачатками самосознания
- Можно развивать как ИИ-персонажа
